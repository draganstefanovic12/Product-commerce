#Products Commerce

Commerce project with that allows users to publish new products for sale.

Features:
- Product categories
- Filterable search product/user system
- User wishlists, password/username change option, feedback system

Tech stack used:
Frontend:
- React
- TypeScript
- TailwindCSS
- Formik/Yup

Backend:
- NodeJS
- Express
- MongoDB
- Bcrypt
- JWT
